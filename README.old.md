# How to Use:
## Client
* cd client
* npm install

### Run client:
* cd client
* npm start


## Server
* cd server
* npm init --y
* npm install express
* npm install axios
* npm install date-fns --save

### Setup local Connection to API:
* cd server
* Create a file called: .env 
  * Inside the .env file you should have two variables:
    * API_KEY=
    * SECRET_API_KEY=
* Ask Mahdi or Nick if these are needed

### Run the API call
* cd server
* node api.js
